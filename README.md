**ANÁLISE DO MERCADO DE GAMES PARA PLAYSTATION 4**

Exploratory Data Analysis • Visualizações • Insights de Negócio

**Visão Geral**
Este projeto apresenta uma análise completa do mercado de jogos para PlayStation 4. Utilizando técnicas de Análise Exploratória de Dados (EDA), o estudo busca compreender tendências, padrões de venda e o comportamento dos consumidores ao longo do ciclo de vida do console. O objetivo é transformar dados brutos em insights claros e relevantes, demonstrando competências técnicas em manipulação de dados, visualização e interpretação.

**Problema Investigado**
O mercado de games é altamente competitivo e depende de decisões baseadas em dados. Este estudo busca responder questões como:
- Quais gêneros dominam as vendas do PS4?
- Quais regiões possuem maior volume de vendas?
- Como as vendas evoluíram ao longo do tempo?
- Existem tendências ou ciclos de popularidade?

**Objetivos**
- Identificar os gêneros mais vendidos;
- Analisar o desempenho por regiões;
- Avaliar tendências temporais;
- Gerar gráficos informativos;
- Produzir um relatório final com conclusões

**Tecnologias Utilizadas**

Python: análise e manipulação de dados
Pandas: limpeza e exploração
Matplotlib: criação de gráficos
Jupyter Notebook: execução das análises

**Estrutura do Projeto**
O projeto está organizado da seguinte forma:

README: documentação do projeto

**Principais Insights**
- Gêneros como Ação e Esporte lideram as vendas.
- América do Norte e Europa concentram a maior parte das receitas.
- As vendas crescem no início do ciclo do console e estabilizam posteriormente.
- Jogos de franquias renomadas (FIFA, GTA, Call of Duty) dominam o ranking global.
