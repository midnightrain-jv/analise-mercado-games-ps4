# Análise do Mercado de Games para PlayStation 4

Exploratory Data Analysis • Visualizações • Insights de Negócio

## Visão geral

Este projeto apresenta uma análise exploratória do mercado de jogos para PlayStation 4, investigando tendências de vendas, gêneros, produtores e regiões. O objetivo é transformar dados brutos em insights claros e relevantes, demonstrando competências em manipulação de dados, visualização e interpretação.

## Problema investigado

O mercado de games é altamente competitivo e depende de decisões baseadas em dados. Este estudo busca responder perguntas como:

- Quais gêneros dominam as vendas do PS4?
- Quais regiões possuem maior volume de vendas?
- Como as vendas evoluíram ao longo do tempo?
- Existem tendências ou ciclos de popularidade?

## Objetivos

- Identificar os gêneros mais vendidos.
- Analisar o desempenho por regiões.
- Avaliar tendências temporais de vendas.
- Gerar gráficos informativos.
- Produzir um relatório final com conclusões.

## Tecnologias utilizadas

- Python: análise e manipulação de dados.
- Pandas: limpeza e exploração.
- Matplotlib: criação de gráficos.
- Jupyter Notebook: execução das análises.
- Git e GitHub: versionamento e organização do projeto.

## Estrutura do projeto

- `data/` – contém os dados originais (`PS4_GamesSales.csv`).
- `notebooks/` – contém os notebooks utilizados para as análises.
- `reports/` – contém gráficos e relatórios gerados (`relatorio_completo.pdf`).
- `.gitignore` – arquivos ignorados pelo Git.
- `README.md` – documentação do projeto.

## Principais insights

- Gêneros como Ação e Esporte lideram as vendas.
- América do Norte e Europa concentram a maior parte das receitas.
- As vendas crescem no início do ciclo do console e se estabilizam posteriormente.
- Jogos de franquias renomadas (por exemplo, FIFA, GTA, Call of Duty) dominam o ranking global.
